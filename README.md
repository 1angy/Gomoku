# 原生 H5+javascript 五子棋小游戏
---
使用H5原生canvas标签配合javascript实现的人机对战五子棋

 - 1 实现了人机对战输赢的判断
 
 - 2 进行了对战的分数计算--(默认未显示在界面，如有需要自行添加即可 computerSource：电脑分数，mySource：我方分数)
 
 - 3 棋子颜色和棋盘大小可随意更改，但请保证尺寸合适

 - 4 纯属学习娱乐，代码写的不好不规范，请谅解
